Ticket Search

This will be 2 search fields.

  • Last name
  • Ticket number

Last name will pull up a table showing all entries with that last name. Data will show: First name, Last name, email address, ticket#. The ticket# will be a link taking you to the registration page, bring all that data over.

The ticket number will bring up a table showing just the info tied to that ticket number: First name, Last name, email address, ticket#. The ticket# will be a link taking you to the registration page, bring all that data over.
